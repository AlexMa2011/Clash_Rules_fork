# Pac

>  Shadowsocks pac file

## 背景

因为部分网站没有默认开启代理，所以新增一部分自定义的 rule。



## 方法

将 pac.txt/user-rule.txt 任意一个替换 Shadowsocks 本地文件夹中的同名文件即可。
